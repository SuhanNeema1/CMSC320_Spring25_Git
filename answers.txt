Q1: Why might we consider using Git?

Answer: Git allows us to manage different version of a codebase, allowing us to collaborate with others, work on separate components of code, and view past versions in a controlled way.

Q3: What is the difference between Merging and Rebasing?

Answer: Merging combines the histories of some Branch A and the main branch in Branch A into a new commit, while rebasing would take all the commits from main onto the latest committed version of the Branch A.
